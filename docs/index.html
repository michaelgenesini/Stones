Grid Example

Features

Responsive

Think Mobile, do Responsive. Stones allows you to quickly create a responsive grid for all kinds of screens, but mobile first.

Lightweight on demand

Stones is modular. You can add to your project just the grid system, or import all modules you need.

Flexible columns

Ok 12 columns layout is good. But if you need somthing different? With Stones and SASS you can set how many grid you want and compile your custom layout.

Your own syntax

row? column? col-sm-2? pure-g-1-3? Yes, they are really cool, but with Stones you can create a grid system based on your thought.

no need to put container everywhere

if you dont put .containerthe content will be fluid for all devices
